1. 绑定IP 和 域名 
		hosts文件
		位置: C:\Windows\System32\drivers\etc\hosts

		内容如下:
			127.0.0.1    www.s86.com
	
	2. 开启虚拟域名
		httpd.conf
		位置: D:\wamp\bin\apache\apache2.4.18\conf\httpd.conf

		1) 按 Ctrl + F 打开搜索
		2) 输入  LoadModule vhost_alias_module modules/mod_vhost_alias.so
		3) 确保 该行配置的最前面没有 # 号

	
	3. 配置虚拟域名
		httpd-vhosts.conf
		位置: D:\wamp\bin\apache\apache2.4.18\conf\extra\httpd-vhosts.conf

		# 访问 www.s86.com  就打开的是   D:/wamp/www/s86/project 目录
		#
		# 注意(重点, 重点, 重点):
		#  以 www.s86.com 访问project目录, 此时的  / 就不再代表www了, 而代表project目录

		
			ServerName www.s86.com
			DocumentRoot D:/wamp/www/s86/project
			
				Options +Indexes +FollowSymLinks +MultiViews
				AllowOverride All
				Require all granted
			
		

	4. 重新启动wamp
		修改了任意配置, 都只会在 重新服务之后才会起作用
